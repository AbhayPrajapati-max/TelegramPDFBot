# Telegram bot for PDF generation

A telegram bot build using Python that convert Text messages to PDF or 
docx file to PDF file

# Features

-> Text to Pdf
-> Word docx file to Pdf
-> Hindi & English supported
-> Automatic file cleanup
-> Telegram bot Intregration

## Installation
''' bash git clone https://github.com/AbhayPrajapati-max/TelegramPDFBot.git
cd TelegramPDFBot
pip install -r Requirements.txt
'''
